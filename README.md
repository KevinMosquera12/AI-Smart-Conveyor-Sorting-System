# Banda-Transportadora-Inteligente-Sistema-de-Clasificaci-n-por-Color-Potenciado-por-IA
Este proyecto implementa una banda transportadora inteligente capaz de clasificar objetos por color en tiempo real. Utiliza un modelo de Machine Learning (KNN) para la detección mediante visión artificial y una ESP32 para la ejecución de movimientos físicos y visualización de datos.
